# Android-leading-css3
使用css3实现手机安卓动态加载动画特效

> 参照 YouTube 实现逻辑

#### 修改布局
> 默认显示水平布局，要想垂直布局请在`.paper-spinner`后加上class `column`
``` html
<div class="paper-spinner column">
  ...
</div>
```

#### 修改颜色
``` css
.paper-spinner {
    color: '自定义颜色'
  }
```
